### 原因 页面右侧滚动条缺失，鼠标滚轮不生效
### 影响 允许 body 垂直滚动；页面统一承担纵向滚动，横向滚动保持在主容器中
### 验证 npm run lint 通过；在水平/垂直布局下手动验证滚轮与滚动条表现

## 变动 保留横向滚动并允许页面纵向增长
### 原因 h-screen 固定视口高度 + 容器 overflow-y-hidden 导致纵向滚动被抑制
### 影响 外层容器改为 min-h-screen；水平布局移除 overflow-y-hidden 让纵向溢出由页面承载
### 验证 在水平布局下横向滚动条可见；页面右侧纵向滚动条可见，滚轮滚动正常
